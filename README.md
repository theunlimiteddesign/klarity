# Klarity

AI-powered career discovery for the next generation.

Klarity helps 15–25 year olds find their ideal career path through 35 adaptive questions covering personality, values, skills, and future trends. Results are powered by AI analysis.

## Features
- 35 adaptive career assessment questions
- AI-powered personality profile
- 12 career paths ranked by match %
- Freemium model — basic results free, full report unlocked
